# DLZ-DB 数据库框架帮助文档

## 目录

DLZ 数据库框架是一个简洁、高效的Java数据库操作框架，提供了面向对象的数据库操作API。它支持链式调用、类型安全的条件构建以及丰富的数据库操作功能。

## 快速导航

- [返回上级目录](../README.md)
- [查看框架源码](../src/main/java/com/dlz/framework/db/modal/DB.java)

1）不需要xml，dao,service，跟mybatis(plus)相比明显简单。并且支持mybatis-plus的大多数能力
2）支持bean与table映射，支持lamda表达式设置字段，仿mybatis-plus，更轻量，使用更简单
3）支持mybatis 的sql语句预设，不需要xml,支持数据库中在线编制。支持 #{id}方式sql预处理，防止sql注入。支持${sql} sql内容替换，可能sql注入，支持预设sql嵌套
4）支持jdbc原生预处理占位符 ？，防止sql注入，支持快速执行sql
5) 执行结果支持bean,支持count,page,ResultMap 继承自JSONMap。取值支持万能转换
6) 支持多数据源
